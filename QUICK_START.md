## 🚀 Getting Started

### For Students
1. Open `index.html` in your browser
2. Choose a course to explore
3. Follow lessons in order from the sidebar
4. Practice with exercises
5. Use sidebar for easy navigation

### For Teachers/Maintainers
1. Each course is self-contained in `courses/COURSE_NAME/`
2. Lessons are in `lessons/` subfolder
3. Exercises are in `exercises/` subfolder
4. Easy to add new content by following existing structure

## 🔧 How to Add New Content

### Adding a New Lesson
1. Create new file: `courses/php/lessons/lesson15.html`
2. Copy template from existing lesson
3. Update navigation links
4. Add to sidebar in `index.html`

### Adding a New Exercise
1. Create new file: `courses/php/exercises/exercise25.html`
2. Copy template from existing exercise
3. Add to exercise index
4. Update navigation

### Adding a New Course
1. Create folder: `courses/NEW_COURSE/`
2. Add subfolders: `lessons/` and `exercises/`
3. Create `index.html` with course overview
4. Add course card to main `index.html`
