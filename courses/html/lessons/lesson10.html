📝 Lesson 10: HTML Forms

Learn how to collect user input with HTML forms. Forms are essential for user interaction, allowing visitors to send information, register, login, and more!

What are HTML Forms?

HTML forms are used to collect user input. Forms can contain different types of input elements like text fields, checkboxes, radio buttons, submit buttons, and more.

Basic Form Structure:

<form action="/submit" method="post">...</form>

Container for all form elements

<label for="id">Label Text</label>

Creates a label for an input field

<input type="text" name="field">

Creates an input field (various types)

<button type="submit">Submit</button>

Creates a submit button

🎯 Key Points

  • Forms are created with <form> tag
  • action attribute specifies where to send data
  • method attribute specifies how to send data (GET or POST)
  • Always use <label> tags for accessibility
  • The name attribute is required to send data to server
  • Use appropriate input types for better user experience

🎯 Progressive Activities

Complete these activities to master HTML forms!

Activity 1

Create Your First Text Input

Beginner

Create a simple form with a text input field for entering a name.

Instructions:

  1. Create a <form> element
  2. Add a <label> with text "Your Name:"
  3. Add an <input> with type="text"
  4. Use id="name" and name="name" on the input
  5. Connect label and input using for="name"
<html>
<body>

<form>
    <label for="name">Your Name:</label><br>
    <input type="text" id="name" name="name">
</form>

</body>
</html>

Expected Output:

A form with a label "Your Name:" and a text input box where users can type.

💡 Hints:

  • The id attribute links the label to the input
  • name attribute is needed when sending data to a server
  • Use <br> to put input on a new line
Activity 2

Add Different Input Types

Beginner

Learn to use different input types like email, password, and number.

Instructions:

  1. Create a form with three fields:
  2. Email field with type="email"
  3. Password field with type="password"
  4. Age field with type="number"
  5. Add proper labels for each field
  6. Add a submit button at the end
<html>
<body>

<form>
    <label for="email">Email:</label><br>
    <input type="email" id="email" name="email"><br><br>
    
    <label for="password">Password:</label><br>
    <input type="password" id="password" name="password"><br><br>
    
    <label for="age">Age:</label><br>
    <input type="number" id="age" name="age" min="1" max="120"><br><br>
    
    <button type="submit">Submit</button>
</form>

</body>
</html>

Expected Output:

A form with three fields: email (validates email format), password (hides input), and age (only allows numbers).

💡 Hints:

  • type="email" validates email format automatically
  • type="password" hides the typed characters
  • type="number" accepts only numbers
  • Use min and max to set number limits
Activity 3

Use Radio Buttons

Intermediate

Learn to create radio buttons that let users select ONE option from a group.

Instructions:

  1. Create a form asking for gender
  2. Add two radio buttons: Male and Female
  3. Both radio buttons must have the same name attribute
  4. Each should have a different value
  5. Add labels for each radio button
<html>
<body>

<form>
    <p>Select Your Gender:</p>
    
    <input type="radio" id="male" name="gender" value="male">
    <label for="male">Male</label><br>
    
    <input type="radio" id="female" name="gender" value="female">
    <label for="female">Female</label><br>
    
    <input type="radio" id="other" name="gender" value="other">
    <label for="other">Other</label>
</form>

</body>
</html>

Expected Output:

Three radio buttons where only ONE can be selected at a time.

💡 Hints:

  • All radio buttons in a group must have the same name attribute
  • This ensures only one can be selected at a time
  • Each option should have a unique id and value
Activity 4

Add Checkboxes

Intermediate

Learn to create checkboxes that let users select MULTIPLE options.

Instructions:

  1. Create a form asking for hobbies
  2. Add at least 4 checkboxes for different hobbies
  3. All can have the same name or different names
  4. Add labels for each checkbox
<html>
<body>

<form>
    <p>Select Your Hobbies:</p>
    
    <input type="checkbox" id="reading" name="hobby" value="reading">
    <label for="reading">Reading</label><br>
    
    <input type="checkbox" id="sports" name="hobby" value="sports">
    <label for="sports">Sports</label><br>
    
    <input type="checkbox" id="music" name="hobby" value="music">
    <label for="music">Music</label><br>
    
    <input type="checkbox" id="gaming" name="hobby" value="gaming">
    <label for="gaming">Gaming</label>
</form>

</body>
</html>

Expected Output:

Four checkboxes where users can select MULTIPLE options at the same time.

💡 Hints:

  • Unlike radio buttons, checkboxes allow multiple selections
  • Users can check or uncheck any number of boxes
  • Checkboxes with the same name can send multiple values
Activity 5

Create a Dropdown List

Intermediate

Learn to create dropdown lists using <select> and <option> tags.

Instructions:

  1. Create a form asking for favorite subject
  2. Use <select> tag with id and name
  3. Add at least 5 <option> tags for different subjects
  4. Each option should have a value attribute
  5. Add a label before the select
<html>
<body>

<form>
    <label for="subject">Favorite Subject:</label><br>
    <select id="subject" name="subject">
        <option value="">-- Select a Subject --</option>
        <option value="math">Mathematics</option>
        <option value="science">Science</option>
        <option value="english">English</option>
        <option value="history">History</option>
        <option value="art">Art</option>
    </select>
</form>

</body>
</html>

Expected Output:

A dropdown menu that opens when clicked, showing all subject options.

💡 Hints:

  • The <select> tag creates the dropdown
  • Each <option> is a choice in the dropdown
  • First option is often a placeholder with empty value
Activity 6

Add a Textarea

Intermediate

Learn to create multi-line text input using <textarea>.

Instructions:

  1. Create a form asking for comments or feedback
  2. Use <textarea> tag with id and name
  3. Set rows="5" and cols="40"
  4. Add a label before the textarea
  5. Add a submit button
<html>
<body>

<form>
    <label for="comments">Your Comments:</label><br>
    <textarea id="comments" name="comments" rows="5" cols="40" 
              placeholder="Enter your feedback here..."></textarea><br><br>
    
    <button type="submit">Submit Feedback</button>
</form>

</body>
</html>

Expected Output:

A large text box where users can type multiple lines of text.

💡 Hints:

  • <textarea> is for multi-line text (unlike <input>)
  • rows sets the height, cols sets the width
  • placeholder shows hint text
Activity 7

Build a Complete Registration Form

Advanced

Create a comprehensive student registration form using all form elements you've learned.

Instructions:

  1. Create a form with action="/register" and method="post"
  2. Add text inputs for: First Name, Last Name
  3. Add email input with required attribute
  4. Add number input for Age
  5. Add date input for Date of Birth
  6. Add radio buttons for Gender
  7. Add dropdown for Grade Level (9th, 10th, 11th, 12th)
  8. Add checkboxes for Course Selection (at least 3 courses)
  9. Add textarea for Special Requests
  10. Add submit button with text "Register"
<html>
<body>

<h2>Student Registration Form</h2>

<form action="/register" method="post">
    <!-- Personal Information -->
    <h3>Personal Information</h3>
    
    <label for="fname">First Name:</label><br>
    <input type="text" id="fname" name="fname" required><br><br>
    
    <label for="lname">Last Name:</label><br>
    <input type="text" id="lname" name="lname" required><br><br>
    
    <label for="email">Email:</label><br>
    <input type="email" id="email" name="email" required><br><br>
    
    <label for="age">Age:</label><br>
    <input type="number" id="age" name="age" min="13" max="19"><br><br>
    
    <label for="dob">Date of Birth:</label><br>
    <input type="date" id="dob" name="dob"><br><br>
    
    <!-- Gender -->
    <p>Gender:</p>
    <input type="radio" id="male" name="gender" value="male">
    <label for="male">Male</label><br>
    <input type="radio" id="female" name="gender" value="female">
    <label for="female">Female</label><br>
    <input type="radio" id="other" name="gender" value="other">
    <label for="other">Other</label><br><br>
    
    <!-- Grade Level -->
    <label for="grade">Grade Level:</label><br>
    <select id="grade" name="grade">
        <option value="">-- Select Grade --</option>
        <option value="9">9th Grade</option>
        <option value="10">10th Grade</option>
        <option value="11">11th Grade</option>
        <option value="12">12th Grade</option>
    </select><br><br>
    
    <!-- Course Selection -->
    <p>Select Courses:</p>
    <input type="checkbox" id="math" name="course" value="math">
    <label for="math">Mathematics</label><br>
    <input type="checkbox" id="science" name="course" value="science">
    <label for="science">Science</label><br>
    <input type="checkbox" id="english" name="course" value="english">
    <label for="english">English</label><br>
    <input type="checkbox" id="history" name="course" value="history">
    <label for="history">History</label><br><br>
    
    <!-- Special Requests -->
    <label for="requests">Special Requests or Comments:</label><br>
    <textarea id="requests" name="requests" rows="4" cols="50"></textarea><br><br>
    
    <!-- Submit Button -->
    <button type="submit">Register</button>
</form>

</body>
</html>

Expected Output:

A complete, professional-looking registration form with all types of input fields organized logically.

🏆 Challenge:

  • Add placeholder attributes to text inputs
  • Group sections using <fieldset> and <legend>
  • Add a "Reset" button with type="reset"
  • Add maxlength attributes to limit text length

📚 Form Elements Reference

Element Description
<form> Defines an HTML form
<input> Defines an input field
<label> Defines a label for an input
<select> Defines a dropdown list
<option> Defines an option in a dropdown
<textarea> Defines a multi-line text input
<button> Defines a clickable button
<fieldset> Groups related form elements
<legend> Defines a caption for fieldset

Input Types

Type Description
text Single-line text input
password Password field (hides input)
email Email address input
number Numeric input
date Date picker
radio Radio button (select one)
checkbox Checkbox (select multiple)
submit Submit button
