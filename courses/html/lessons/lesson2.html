🏷️ Lesson 2: HTML Elements & Tags

Learn about tags, elements, and how they work together to build web pages

📚 What is a Tag?

An HTML tag is a keyword surrounded by angle brackets < >. Tags are the building blocks of HTML - they tell the browser what kind of content to display.

<tagname>
🎯 Key Points About Tags:
  • Tags are always written inside angle brackets: < and >
  • Most tags come in pairs: an opening tag and a closing tag
  • The closing tag has a forward slash / before the tag name
  • Tags are not case sensitive, but we always use lowercase
  • Examples: <h1>, <p>, <div>, <a>, <img>

Opening Tag vs Closing Tag:

<h1> ← Opening Tag
</h1> ← Closing Tag (notice the / slash)

📦 What is an Element?

An HTML element is the complete package: the opening tag + content + closing tag. Everything together forms an element!

<h1> Hello World </h1>

↑ This entire thing is an HTML element

Parts of an Element:

Part Example Purpose
Opening Tag <h1> Marks where the element starts
Content Hello World The actual text or data displayed
Closing Tag </h1> Marks where the element ends

More Examples:

<p>This is a paragraph element.</p>

<h2>This is a heading element.</h2>

<strong>This text will be bold.</strong>

Nested Elements:

Elements can be placed inside other elements (this is called nesting):

<body>
    <h1>My Website</h1>
    <p>This is a <strong>paragraph</strong> with bold text.</p>
</body>
⚠️ Important Rule:
  • Always close your tags! Every opening tag needs a closing tag.
  • Close tags in the correct order (last opened, first closed)

Empty Elements (Self-Closing Tags):

Some elements don't have content or closing tags. These are called empty elements:

Element Purpose Example
<br> Line break First line<br>Second line
<hr> Horizontal line/divider <hr>
<img> Display an image <img src="photo.jpg">

🎯 Practice Activity

1

Create a Page Using Different Elements Beginner

Goal: Practice using different HTML elements with proper opening and closing tags.

📝 Instructions:

  1. Create a new file called elements-practice.html
  2. Set up the basic HTML structure (html, head, title, body)
  3. Inside the body, create:
    • An h1 heading with your favorite animal
    • A paragraph (p) describing why you like that animal
    • An h2 heading that says "Fun Facts"
    • Another paragraph with 2-3 fun facts
    • Use <strong> tags to make important words bold
    • Use <br> tags to create line breaks between facts
  4. Make sure every opening tag has a closing tag!
  5. Open your file in a browser to see the result
<html>
<head>
<title>My Favorite Animal</title>
</head>
<body>
<h1>Dolphins</h1>
<p>I love dolphins because they are <strong>intelligent</strong> and friendly!</p>
<h2>Fun Facts</h2>
<p>
Dolphins can swim up to <strong>20 miles per hour</strong>!<br>
They communicate using clicks and whistles.<br>
Dolphins sleep with one eye open!
</p>
</body>
</html>

✅ Expected Output:

Your page should display:

Dolphins

I love dolphins because they are intelligent and friendly!

Fun Facts

Dolphins can swim up to 20 miles per hour!
They communicate using clicks and whistles.
Dolphins sleep with one eye open!

💡 Tips & Reminders
  • Opening vs Closing: Remember <p> opens, </p> closes
  • <strong> tag: Makes text bold/important
  • <br> tag: Creates a line break (no closing tag needed!)
  • Nesting: Tags inside other tags must be closed in reverse order
  • Indentation: Use spaces or tabs to make your code readable

🚀 Try More:

  • Add an <hr> tag to create a horizontal line between sections
  • Use <em> tags to make some text italic (e.g., <em>text</em>)
  • Try nesting <strong> inside <em> to make text bold AND italic!
  • Create another section about a different animal
