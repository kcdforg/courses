🌐 Lesson 1: Introduction to HTML

Your first step into web development - learn the building blocks of the web!

📚 What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It's like the skeleton of a website - it provides the structure and content that browsers display.

🎯 Key Points:
  • HTML is a markup language, not a programming language
  • HTML uses tags (like <h1>, <p>, <div>) to structure content
  • Browsers (Chrome, Firefox, Safari) read HTML and display web pages
  • Every website you visit is built with HTML
  • HTML is easy to learn - perfect for beginners!

Basic HTML Structure:

Every HTML page has a simple structure with these essential parts:

<html>
<head>
    <title> My First Web Page</title>
</head>
<body>
    <h1> This is a Heading tag</h1>
    <p> This is a paragraph tag </p>
</body>
</html>

Understanding HTML Tags:

Tag Purpose Example
<html> Root element - wraps the entire page <html>...</html>
<head> Contains information about the page (not visible) <head>...</head>
<title> Sets the page title (shown in browser tab) <title>My Website</title>
<body> Contains all visible content <body>...</body>
<h1> Main heading (largest) <h1>Welcome!</h1>
<p> Paragraph of text <p>Hello World!</p>
