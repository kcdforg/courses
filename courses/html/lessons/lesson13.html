🎬 Lesson 13: HTML5 Multimedia

Learn to embed videos, audio, and external content like YouTube videos and Google Maps directly into your web pages using HTML5 multimedia elements!

What are HTML5 Multimedia Elements?

HTML5 introduced native support for multimedia content without needing plugins like Flash. You can now easily embed videos, audio files, and external content directly in your web pages.

Main Multimedia Elements:

<video>...</video>

Embeds video files directly in your webpage

<audio>...</audio>

Embeds audio files for music or sounds

<iframe>...</iframe>

Embeds external content (YouTube, maps, other websites)

<source>

Specifies multiple file formats for compatibility

🎯 Key Points

  • HTML5 multimedia works without Flash or other plugins
  • Always provide multiple formats for browser compatibility
  • Use controls attribute to show play/pause buttons
  • Avoid autoplay - it can be annoying to users
  • Iframes can embed external content from other websites

🎯 Progressive Activities

Complete these activities to master HTML multimedia!

Activity 1

Embed a Video

Beginner

Learn to embed a video file using the <video> element.

Instructions:

  1. Create a <video> element
  2. Add width="320" and height="240" attributes
  3. Add the controls attribute to show play/pause buttons
  4. Inside video, add a <source> with src="video.mp4"
  5. Add fallback text for browsers that don't support video
<html>
<body>

<h2>My First Video</h2>

<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

</body>
</html>

Expected Output:

A video player with play/pause controls. (Note: You need an actual video file for it to work)

💡 Hints:

  • controls attribute adds play, pause, and volume controls
  • The fallback text shows if the browser doesn't support video
  • type="video/mp4" tells the browser the file format
Activity 2

Add an Audio Player

Beginner

Learn to embed an audio file using the <audio> element.

Instructions:

  1. Create an <audio> element with controls
  2. Add a <source> with src="song.mp3"
  3. Specify type="audio/mpeg"
  4. Add fallback text for unsupported browsers
<html>
<body>

<h2>Listen to This Song</h2>

<audio controls>
    <source src="song.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

</body>
</html>

Expected Output:

An audio player with play/pause controls. (Note: You need an actual audio file for it to work)

💡 Hints:

  • <audio> works similarly to <video>
  • Audio player is smaller and only shows controls, no video
  • MP3 format works in all modern browsers
Activity 3

Video with Multiple Formats

Intermediate

Learn to provide multiple video formats for better browser compatibility.

Instructions:

  1. Create a video element with controls
  2. Add TWO <source> elements
  3. First source: "video.mp4" with type="video/mp4"
  4. Second source: "video.webm" with type="video/webm"
  5. Browser will use the first format it supports
<html>
<body>

<h2>Video with Multiple Formats</h2>

<video width="640" height="360" controls>
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    Your browser does not support the video tag.
</video>

</body>
</html>

Expected Output:

A video player that works across all browsers by trying different formats.

💡 Hints:

  • The browser tries each source in order
  • It uses the first format it can play
  • MP4 works in all modern browsers, WebM is a backup
Activity 4

Video with Advanced Attributes

Intermediate

Learn to use additional video attributes like poster, loop, and muted.

Instructions:

  1. Create a video with controls
  2. Add poster="thumbnail.jpg" (image shown before playing)
  3. Add loop attribute (video repeats)
  4. Add muted attribute (starts without sound)
  5. Set width and height
<html>
<body>

<h2>Video with All Features</h2>

<video width="640" height="360" controls loop muted 
       poster="thumbnail.jpg">
    <source src="video.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

</body>
</html>

Expected Output:

A video player showing a thumbnail image before playing, muted by default, and repeating when finished.

💡 Hints:

  • poster shows an image before the video starts
  • loop makes the video repeat automatically
  • muted starts with no sound (unmute with controls)
  • Avoid using autoplay - it annoys users!
Activity 5

Embed a YouTube Video

Intermediate

Learn to embed YouTube videos using the <iframe> element.

Instructions:

  1. Create an <iframe> element
  2. Set width="560" and height="315"
  3. Set src to a YouTube embed URL
  4. Add allowfullscreen attribute
  5. Tip: Get embed code from YouTube's "Share" button
<html>
<body>

<h2>Watch This Tutorial</h2>

<iframe width="560" height="315" 
        src="https://www.youtube.com/embed/dQw4w9WgXcQ" 
        allowfullscreen>
</iframe>

<p>Note: Replace the video ID in the URL with your chosen video.</p>

</body>
</html>

Expected Output:

A YouTube video player embedded directly in your page with full controls.

💡 Hints:

  • Go to any YouTube video and click "Share" → "Embed"
  • Copy the iframe code provided by YouTube
  • The video ID is the last part of the URL
  • allowfullscreen enables fullscreen button
Activity 6

Create a Multimedia Page

Advanced

Build a complete multimedia page with video, audio, and embedded content.

Instructions:

  1. Create a page with proper structure (header, main, sections)
  2. Section 1: Video player with poster and controls
  3. Section 2: Audio player with background music
  4. Section 3: Embedded YouTube video
  5. Section 4: Optional - Google Maps iframe of your location
  6. Add headings and descriptions for each section
<html>
<body>

<header>
    <h1>My Multimedia Gallery</h1>
    <p>Explore videos, music, and more!</p>
</header>

<main>
    <section>
        <h2>Featured Video</h2>
        <video width="640" height="360" controls poster="poster.jpg">
            <source src="featured.mp4" type="video/mp4">
            Your browser does not support video.
        </video>
        <p>Watch our latest video content!</p>
    </section>

    <section>
        <h2>Background Music</h2>
        <audio controls>
            <source src="music.mp3" type="audio/mpeg">
            Your browser does not support audio.
        </audio>
        <p>Enjoy our curated playlist.</p>
    </section>

    <section>
        <h2>Tutorial Video</h2>
        <iframe width="560" height="315"
                src="https://www.youtube.com/embed/dQw4w9WgXcQ"
                allowfullscreen>
        </iframe>
        <p>Learn from expert tutorials on YouTube.</p>
    </section>

    <section>
        <h2>Find Us Here</h2>
        <iframe 
            src="https://www.google.com/maps/embed?pb=!1m18!..." 
            width="600" 
            height="450" 
            style="border:0;" 
            allowfullscreen>
        </iframe>
        <p>Visit our location on Google Maps.</p>
    </section>
</main>

<footer>
    <p>© 2025 Multimedia Gallery</p>
</footer>

</body>
</html>

Expected Output:

A complete multimedia page with multiple sections, each containing different types of media content.

🏆 Challenge:

  • Add CSS styling to make sections look professional
  • Create a grid layout for multiple videos
  • Add a playlist of audio files
  • Embed a Spotify playlist using iframe

📚 Multimedia Reference

Video Attributes

Attribute Description
controls Show play/pause/volume controls
autoplay Start playing automatically (avoid this!)
loop Repeat video when finished
muted Start with audio muted
poster Image shown before video plays
width/height Video player dimensions

Supported Formats

Video: MP4 (best compatibility), WebM, Ogg

Audio: MP3 (best compatibility), WAV, Ogg
