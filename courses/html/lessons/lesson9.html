📊 Lesson 9: HTML Tables

Learn how to create tables to organize and display data in rows and columns. Tables are perfect for showing schedules, comparisons, and structured information!

What are HTML Tables?

HTML tables allow you to organize data into rows and columns. Tables are useful for displaying structured information like schedules, price lists, comparison charts, and more.

Table Tags:

<table>...</table>

Defines the entire table

<tr>...</tr>

Defines a table row (table row)

<th>...</th>

Defines a table header cell (bold and centered)

<td>...</td>

Defines a table data cell (table data)

🎯 Key Points

  • Tables organize data in rows and columns
  • Each row is created with <tr>
  • Header cells use <th> (bold and centered)
  • Data cells use <td> (regular text)
  • Tables should be used for data, not page layout

📋 Table Structure:

<table border="1">
<tr>
<th>Header 1</th>
<th>Header 2</th>
</tr>
<tr>
<td>Data 1</td>
<td>Data 2</td>
</tr>
</table>

🎯 Progressive Activities

Complete these activities to master HTML tables!

Activity 1

Create Your First Table

Beginner

Create a simple table showing student information with Name, Age, and Grade columns.

Instructions:

  1. Create a <table> with border="1"
  2. Add a header row with <th> for "Name", "Age", and "Grade"
  3. Add 2 data rows with information about students
  4. Use <tr> for rows and <td> for data cells
<html>
<body>

<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>Grade</th>
    </tr>
    <tr>
        <td>John</td>
        <td>15</td>
        <td>10th</td>
    </tr>
    <tr>
        <td>Sarah</td>
        <td>14</td>
        <td>9th</td>
    </tr>
</table>

</body>
</html>

Expected Output:

A table with headers (Name, Age, Grade) and two rows of student data with visible borders.

💡 Hints:

  • The border="1" attribute makes borders visible
  • <th> cells are automatically bold and centered
  • Each row must have the same number of cells
Activity 2

Add a Caption to Your Table

Beginner

Learn to add a title to your table using the <caption> tag.

Instructions:

  1. Create a table showing subjects and scores
  2. Add a <caption> tag right after <table>
  3. The caption should say "My Test Scores"
  4. Add headers for "Subject" and "Score"
  5. Add at least 3 rows of data
<html>
<body>

<table border="1">
    <caption>My Test Scores</caption>
    <tr>
        <th>Subject</th>
        <th>Score</th>
    </tr>
    <tr>
        <td>Math</td>
        <td>95</td>
    </tr>
    <tr>
        <td>Science</td>
        <td>88</td>
    </tr>
    <tr>
        <td>English</td>
        <td>92</td>
    </tr>
</table>

</body>
</html>

Expected Output:

A table with "My Test Scores" as the title above the table, showing subjects and their scores.

💡 Hints:

  • <caption> must come right after <table>
  • Caption appears centered above the table
Activity 3

Use Colspan to Span Columns

Intermediate

Learn to make a cell span across multiple columns using colspan.

Instructions:

  1. Create a table with 3 columns
  2. In the first row, use colspan="3" on a single <th>
  3. The header should say "Student Information"
  4. Second row should have 3 separate headers: Name, Age, Grade
  5. Add a data row with student information
<html>
<body>

<table border="1">
    <tr>
        <th colspan="3">Student Information</th>
    </tr>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>Grade</th>
    </tr>
    <tr>
        <td>Mike</td>
        <td>16</td>
        <td>11th</td>
    </tr>
</table>

</body>
</html>

Expected Output:

A table where "Student Information" spans across all 3 columns in the first row.

💡 Hints:

  • colspan="3" makes a cell span 3 columns
  • When using colspan, don't add extra cells in that row
  • The colspan value should match the number of columns in other rows
Activity 4

Use Rowspan to Span Rows

Intermediate

Learn to make a cell span across multiple rows using rowspan.

Instructions:

  1. Create a class schedule table with 3 columns: Day, Morning, Afternoon
  2. Use rowspan="2" on "Monday" to make it span 2 rows
  3. Add morning and afternoon subjects for both periods
  4. Remember: when using rowspan, skip that cell in the next row!
<html>
<body>

<table border="1">
    <tr>
        <th>Day</th>
        <th>Morning</th>
        <th>Afternoon</th>
    </tr>
    <tr>
        <td rowspan="2">Monday</td>
        <td>Math</td>
        <td>Science</td>
    </tr>
    <tr>
        <td>English</td>
        <td>History</td>
    </tr>
</table>

</body>
</html>

Expected Output:

A table where "Monday" appears in a single cell that spans 2 rows vertically.

💡 Hints:

  • rowspan="2" makes a cell span 2 rows
  • In the second row, don't include the Day cell (it's already spanned)
  • The second row only needs 2 cells (Morning and Afternoon)
Activity 5

Organize with thead, tbody, and tfoot

Intermediate

Learn to organize your table into logical sections: header, body, and footer.

Instructions:

  1. Create a shopping list table with Item, Quantity, and Price columns
  2. Use <thead> for the header row
  3. Use <tbody> for the data rows (at least 2 items)
  4. Use <tfoot> for a total row
  5. In the footer, use colspan="2" for "Total" label
<html>
<body>

<table border="1">
    <thead>
        <tr>
            <th>Item</th>
            <th>Quantity</th>
            <th>Price</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Notebook</td>
            <td>5</td>
            <td>$2.50</td>
        </tr>
        <tr>
            <td>Pen</td>
            <td>10</td>
            <td>$1.00</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td colspan="2">Total</td>
            <td>$22.50</td>
        </tr>
    </tfoot>
</table>

</body>
</html>

Expected Output:

A well-organized table with header, body, and footer sections showing a shopping list with total.

💡 Hints:

  • <thead>, <tbody>, and <tfoot> help organize large tables
  • These tags don't change the visual appearance but improve structure
  • Screen readers use these sections for better navigation
Activity 6

Create a Weekly Schedule

Advanced

Build a complete weekly class schedule using all table features you've learned.

Instructions:

  1. Create a table with a caption "My Class Schedule"
  2. Header row with: Period, Monday, Tuesday, Wednesday, Thursday, Friday
  3. Add 4 rows for Period 1-4
  4. Fill in with subject names
  5. Use colspan for lunch break spanning all days
  6. Add borders to all cells
<html>
<body>

<table border="1">
    <caption>My Class Schedule</caption>
    <thead>
        <tr>
            <th>Period</th>
            <th>Monday</th>
            <th>Tuesday</th>
            <th>Wednesday</th>
            <th>Thursday</th>
            <th>Friday</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Period 1</td>
            <td>Math</td>
            <td>English</td>
            <td>Science</td>
            <td>History</td>
            <td>Art</td>
        </tr>
        <tr>
            <td>Period 2</td>
            <td>Science</td>
            <td>Math</td>
            <td>PE</td>
            <td>English</td>
            <td>Music</td>
        </tr>
        <tr>
            <td>Lunch</td>
            <td colspan="5" style="text-align: center;">Lunch Break</td>
        </tr>
        <tr>
            <td>Period 3</td>
            <td>History</td>
            <td>Science</td>
            <td>Math</td>
            <td>Art</td>
            <td>English</td>
        </tr>
        <tr>
            <td>Period 4</td>
            <td>PE</td>
            <td>History</td>
            <td>English</td>
            <td>Math</td>
            <td>Science</td>
        </tr>
    </tbody>
</table>

</body>
</html>

Expected Output:

A complete weekly schedule with periods as rows, days as columns, and a lunch break spanning all days.

🏆 Challenge:

  • Try adding a <tfoot> row showing "Classes end at 3 PM"
  • Experiment with different colspan values
  • Add a color to the lunch row using inline CSS

📚 Table Tags Reference

Tag Description
<table> Defines a table
<tr> Defines a table row
<th> Defines a table header cell (bold and centered)
<td> Defines a table data cell
<caption> Defines a table caption/title
<thead> Groups the header content
<tbody> Groups the body content
<tfoot> Groups the footer content
colspan Number of columns a cell should span
rowspan Number of rows a cell should span
