🏷️ Lesson 18: ID and Class Attributes

Learn how to identify and group HTML elements for styling and functionality!

📚 What are ID and Class?

ID and Class are special attributes that you add to HTML elements. They help you:

  • Identify specific elements on your page
  • Style elements with CSS
  • Select elements with JavaScript
  • Organize your HTML code
🎯 Key Difference:
  • ID = Unique identifier (like your National ID - only ONE person can have it)
  • Class = Group identifier (like a school class - many students can belong to the same class)

🆔 The ID Attribute

An ID is a unique name for a single element. Each ID can only be used ONCE per page.

<element id="uniqueName">Content</element>

ID Syntax Examples:

<h1 id="main-title">Welcome to My Website</h1>

<div id="header">This is the header section</div>

<p id="intro-paragraph">Introduction text here</p>

📦 The Class Attribute

A Class is a name that can be used on multiple elements. Think of it as a group name.

<element class="groupName">Content</element>

Class Syntax Examples:

<p class="highlight">This is highlighted</p>

<p class="highlight">This is also highlighted</p>

<span class="highlight">Me too!</span>

🎨 Multiple Classes

An element can have multiple classes separated by spaces:

<button class="btn btn-primary">Click Me</button>

<button class="btn btn-success">Submit</button>

<button class="btn btn-danger">Delete</button>

🔄 ID + Class Together

An element can have both an ID and classes:

<div id="main-container" class="container large-box">
    Content here
</div>

📊 ID vs Class - Comparison

🆔 ID

  • Use # in CSS: #myId { }
  • Must be unique on page
  • Only ONE ID per element
  • Higher specificity (stronger)
  • Can link to with #id in URL
  • Good for: headers, main sections, unique elements

📦 Class

  • Use . in CSS: .myClass { }
  • Can be reused on multiple elements
  • Multiple classes per element
  • Lower specificity
  • Cannot link to in URL
  • Good for: styling groups, components, repeated patterns
Feature ID Class
CSS Selector #name .name
Uniqueness Unique (once per page) Reusable (many times)
Per Element Only 1 ID Multiple classes allowed
Real-life example National ID Number School Class (Grade 5)
When to use For unique elements For groups of similar elements

⚠️ Common Mistakes to Avoid:

  • ❌ Using the same ID more than once on a page
  • ❌ Starting ID/class names with numbers (use letters first)
  • ❌ Using spaces in names (use hyphens instead: my-class)
  • ❌ Confusing # (ID) with . (class) in CSS

💡 Naming Best Practices:

  • ✅ Use meaningful names: header, nav-menu, btn-submit
  • ✅ Use lowercase letters
  • ✅ Use hyphens for multiple words: main-content
  • ✅ Be descriptive: product-card is better than card1

Your First ID and Class Beginner

Goal: Create a simple page with ID and class attributes

📝 Instructions:

  1. Create a new file called id-class-practice.html
  2. Add a heading with an ID of "page-title"
  3. Add 3 paragraphs, all with the class "text-content"
  4. Add internal CSS to style them
<html>
<head>
<title>ID and Class Practice</title>
<style>
/* ID uses # symbol */
#page-title {
color: blue;
text-align: center;
}
/* Class uses . symbol */
.text-content {
background-color: lightyellow;
padding: 10px;
margin: 10px;
}
</style>
</head>
<body>
<h1 id="page-title">Welcome to My Page</h1>
<p class="text-content">First paragraph</p>
<p class="text-content">Second paragraph</p>
<p class="text-content">Third paragraph</p>
</body>
</html>

Multiple Classes Intermediate

Goal: Create buttons with multiple classes for different styles

📝 Instructions:

  1. Create a base .btn class for common button styles
  2. Create color classes: .btn-blue, .btn-green, .btn-red
  3. Create size classes: .btn-small, .btn-large
  4. Combine classes on buttons
<style>
/* Base button style */
.btn {
padding: 10px 20px;
border: none;
border-radius: 5px;
cursor: pointer;
color: white;
}
/* Color classes */
.btn-blue { background-color: #667eea; }
.btn-green { background-color: #28a745; }
.btn-red { background-color: #dc3545; }
/* Size classes */
.btn-small { font-size: 12px; padding: 5px 10px; }
.btn-large { font-size: 20px; padding: 15px 30px; }
</style>
<!-- Combining multiple classes -->
<button class="btn btn-blue">Blue Button</button>
<button class="btn btn-green btn-large">Large Green</button>
<button class="btn btn-red btn-small">Small Red</button>

Build a Product Card System Intermediate

Goal: Create a reusable card system using classes, with unique IDs for specific cards

📝 Instructions:

  1. Create a .card class for the basic card style
  2. Create a .card-featured class for special cards
  3. Give each card a unique ID (product-1, product-2, product-3)
  4. Style the featured card differently
<style>
.card {
border: 2px solid #ddd;
padding: 20px;
margin: 10px;
border-radius: 10px;
display: inline-block;
width: 200px;
}
.card-featured {
border-color: gold;
background-color: #fffef0;
box-shadow: 0 0 10px gold;
}
</style>
<div id="product-1" class="card">
<h3>Product 1</h3>
<p>$29.99</p>
</div>
<div id="product-2" class="card card-featured">
<h3>⭐ Featured Product</h3>
<p>$49.99</p>
</div>
<div id="product-3" class="card">
<h3>Product 3</h3>
<p>$19.99</p>
</div>

Create a Navigation Menu Advanced

Goal: Build a navigation menu using ID for the container and classes for menu items

📝 Instructions:

  1. Create a nav with id="main-nav"
  2. Add links with class="nav-link"
  3. Add one link with an additional class="active"
  4. Style the active link differently
<style>
#main-nav {
background-color: #333;
padding: 15px;
}
.nav-link {
color: white;
text-decoration: none;
padding: 10px 20px;
margin: 0 5px;
}
.nav-link.active {
background-color: #667eea;
border-radius: 5px;
}
</style>
<nav id="main-nav">
<a href="#" class="nav-link active">Home</a>
<a href="#" class="nav-link">About</a>
<a href="#" class="nav-link">Services</a>
<a href="#" class="nav-link">Contact</a>
</nav>
