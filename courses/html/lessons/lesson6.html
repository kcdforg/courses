🔗 Lesson 6: HTML Links (<a> Tag)

Learn to create clickable links and connect pages together

📚 Understanding the <a> Tag

The anchor (<a>) tag is used to create hyperlinks. Links are what make the web a "web" - they connect pages together and allow users to navigate between them.

🎯 Key Points:
  • The <a> tag creates clickable links
  • The href attribute specifies the destination URL
  • Links can point to web pages, files, email addresses, or sections within a page
  • The text between the tags is what users see and click
  • Links can be styled and can contain other elements like images

Basic Syntax:

<a href="destination-url">Link Text</a>

Common Attributes:

Attribute Description Example
href Specifies the link destination href="page.html"
target Where to open the link target="_blank" (new tab)
title Tooltip text on hover title="Visit Homepage"
download Download file instead of navigate download="file.pdf"

🎯 Hands-On Activities

1

Create Your First Link Beginner

Goal: Create a simple link that takes users to Google.

📝 Instructions:

  1. Create a new HTML file called links-practice.html
  2. Set up the basic HTML structure (DOCTYPE, html, head, body)
  3. Inside the body, create a link with the text "Visit Google"
  4. The link should point to https://www.google.com
  5. Open the file in your browser and click the link

✅ Expected Output:

You should see a clickable blue underlined text "Visit Google". When clicked, it should take you to Google's homepage.

💡 Need a hint?

Use the <a> tag with the href attribute:

<a href="https://www.google.com">Visit Google</a>
2

Build a Simple Navigation Menu Beginner

Goal: Create a navigation menu with internal links to different pages.

📝 Instructions:

  1. Create four HTML files: home.html, about.html, services.html, and contact.html
  2. In home.html, create a navigation menu with links to all four pages
  3. Add a heading in each page to identify it (e.g., "Home Page", "About Us", etc.)
  4. Copy the same navigation menu to all four pages
  5. Test by clicking through all the links
<!-- Example navigation menu -->
<nav>
<a href="home.html">Home</a> |
<a href="about.html">About</a> |
<a href="services.html">Services</a> |
<a href="contact.html">Contact</a>
</nav>

✅ Expected Output:

A navigation bar with four clickable links separated by "|". Clicking each link should take you to the corresponding page.

🚀 Challenge:

Add inline CSS to style your navigation: change the link colors, remove the underline, and add spacing between links.

3

Create Jump Links (Anchor Links) Intermediate

Goal: Create a long page with a table of contents that jumps to different sections.

📝 Instructions:

  1. Create a file called article.html
  2. Add a "Table of Contents" at the top with links to 4 sections
  3. Create 4 sections with headings: Introduction, Methods, Results, Conclusion
  4. Give each section heading an id attribute
  5. Link each TOC item to its corresponding section using href="#section-id"
  6. Add a "Back to Top" link at the end of each section
<!-- Table of Contents -->
<h2>Table of Contents</h2>
<ul>
<li><a href="#intro">Introduction</a></li>
<li><a href="#methods">Methods</a></li>
</ul>
<!-- Section -->
<h2 id="intro">Introduction</h2>
<p>Your content here...</p>
<a href="#top">Back to Top</a>

✅ Expected Output:

Clicking a TOC link should smoothly scroll to that section. The "Back to Top" links should return you to the top of the page.

💡 Need a hint?
  • Use id="section-name" on the heading elements
  • Use href="#section-name" in your links
  • Add id="top" to an element at the top of the page
  • Or use href="#" to go to the very top

🚀 Challenge:

Add smooth scrolling behavior using CSS: html { scroll-behavior: smooth; }

4

Email and Phone Links Beginner

Goal: Create special links that open email clients and dial phone numbers.

📝 Instructions:

  1. Create a file called contact.html
  2. Add a heading "Contact Us"
  3. Create an email link using mailto: that opens the default email app
  4. Add a pre-filled subject line using ?subject=
  5. Create a phone link using tel: protocol
  6. Test both links
<!-- Email Link -->
<a href="mailto:info@example.com?subject=Inquiry">
Send us an email
</a>
<!-- Phone Link -->
<a href="tel:+1234567890">
Call us: (123) 456-7890
</a>

✅ Expected Output:

Clicking the email link should open your default email client with the email address pre-filled. Clicking the phone link should prompt to dial the number (on mobile) or open a calling app.

🚀 Challenge:

Add multiple recipients, CC, BCC, and body text to your email link:
mailto:email1@example.com,email2@example.com?cc=cc@example.com&bcc=bcc@example.com&subject=Hello&body=Message%20text

Download Links and Target Attribute Intermediate

Goal: Create links that download files and open in new tabs.

📝 Instructions:

  1. Create a folder called downloads
  2. Create a simple text file called readme.txt inside the folder
  3. Create an HTML page called resources.html
  4. Add a link that downloads the readme.txt file using the download attribute
  5. Add external links (to Wikipedia, YouTube, etc.) that open in new tabs using target="_blank"
  6. Add the rel="noopener noreferrer" attribute for security
<!-- Download Link -->
<a href="downloads/readme.txt" download>
📥 Download README
</a>
<!-- External Link in New Tab -->
<a href="https://www.wikipedia.org"
target="_blank"
rel="noopener noreferrer">
Visit Wikipedia (opens in new tab)
</a>

✅ Expected Output:

The download link should trigger a file download. External links should open in new browser tabs.

💡 Why use rel="noopener noreferrer"?

noopener prevents the new page from accessing your page's window object (security).
noreferrer prevents the new page from knowing which page sent the visitor (privacy).

6

Image Links and Button-Style Links Intermediate

Goal: Turn images into clickable links and style links to look like buttons.

📝 Instructions:

  1. Find or create a small image (logo.png or similar)
  2. Create a page called styled-links.html
  3. Make the image clickable by wrapping it in an <a> tag
  4. Create text links styled to look like buttons using inline CSS
  5. Create different button styles: primary (blue), success (green), danger (red)
  6. Add hover effects using the :hover pseudo-class in a <style> section
<!-- Image Link -->
<a href="home.html">
<img src="logo.png" alt="Home" width="100">
</a>
<!-- Button-Style Link -->
<a href="signup.html"
style="background-color: #667eea;
color: white;
padding: 12px 30px;
text-decoration: none;
border-radius: 5px;
display: inline-block;">
Sign Up Now
</a>

✅ Expected Output:

Clicking the image should navigate to a page. Button-style links should look like actual buttons with colored backgrounds and rounded corners.

🚀 Challenge:

Create a card component with an image, title, description, and a "Read More" button link - all styled with CSS!

7

Final Project: Personal Portfolio Page Advanced

Goal: Create a complete multi-page personal portfolio website using all link types.

📝 Project Requirements:

  1. Create 4 pages: index.html (home), about.html, projects.html, contact.html
  2. Navigation: Add a consistent navigation menu on all pages with internal links
  3. Home Page: Include a clickable logo image that links to home
  4. About Page: Create jump links to different sections (Education, Skills, Hobbies)
  5. Projects Page: Display 3 projects with:
    • External links to live demos (target="_blank")
    • Download links for project files
    • Button-styled links for "View Details"
  6. Contact Page: Include:
    • Email link with pre-filled subject
    • Phone link
    • Social media links (GitHub, LinkedIn) opening in new tabs
    • A "Back to Top" link
  7. Styling: Use CSS to style all links, add hover effects, and create button-style call-to-action links

✅ Project Checklist:

  • ☐ All internal navigation works correctly
  • ☐ External links open in new tabs
  • ☐ Jump links scroll to correct sections
  • ☐ Email and phone links work properly
  • ☐ Download links trigger downloads
  • ☐ Image links are clickable
  • ☐ All links have appropriate styling
  • ☐ Links have hover effects
  • ☐ Code is properly indented and organized

🚀 Bonus Challenges:

  • Add a "scroll to top" button that appears when you scroll down
  • Create a breadcrumb navigation showing current page location
  • Add smooth scrolling behavior throughout the site
  • Create an active state for the current page in navigation
  • Make the navigation responsive for mobile devices
