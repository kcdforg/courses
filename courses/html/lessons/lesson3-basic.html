📝 Lesson 3: HTML Basic Tags

Master the essential HTML tags every web developer needs to know

📚 The Essential HTML Tags

In this lesson, you'll learn the most commonly used HTML tags that form the foundation of every web page. These are the tags you'll use in almost every HTML document you create!

🎯 What You'll Learn:
  • How to structure an HTML document properly
  • Creating headings from h1 to h6
  • Adding paragraphs of text
  • Creating clickable links
  • Displaying images on your page
  • Adding line breaks with <br>
  • Creating dividers with <hr>

The Basic HTML Structure:

Every HTML page follows this simple structure:

<html>
<head>
    <title>Page Title</title>
</head>
<body>
    Your visible content goes here!
</body>
</html>

The Essential Tags:

<h1> to <h6>

Headings

Create headings of different sizes. h1 is the largest (most important), h6 is the smallest.

<h1>Main Title</h1>
<h2>Subtitle</h2>
<p>

Paragraph

Display blocks of text. Browsers automatically add space before and after paragraphs.

<p>This is a paragraph.</p>
<a>

Link (Anchor)

Create clickable links to other pages or websites. Uses the href attribute.

<a href="page.html">Click me</a>
<img>

Image

Display images. This is a self-closing tag (no closing tag needed). Uses src attribute.

<img src="photo.jpg" alt="Description">
<br>

Line Break

Creates a line break within text. Self-closing tag (no closing tag needed).

First line<br>
Second line
<hr>

Horizontal Rule

Creates a horizontal line/divider. Self-closing tag. Used to separate content sections.

<hr>

HTML Headings in Detail:

HTML provides 6 levels of headings. Use them to organize your content:

Tag Size When to Use
<h1> Largest Main page title (use only once per page)
<h2> Large Major sections
<h3> Medium-Large Subsections within h2 sections
<h4> Medium Sub-subsections
<h5> Small Minor headings
<h6> Smallest Least important headings
💡 Best Practice:
  • Use headings in order (don't skip levels: h1 → h2 → h3)
  • Use only one h1 per page (for the main title)
  • Headings help organize content and improve accessibility

🎯 Practice Activity

1

Build a Complete Web Page Beginner

Goal: Create a complete web page using all the basic HTML tags you've learned.

📝 Instructions:

  1. Create a new file called my-profile.html
  2. Set up the basic HTML structure (html, head, title, body)
  3. In the title, write "My Profile Page"
  4. Inside the body, create:
    • An h1 heading with your name
    • An h2 heading that says "About Me"
    • A paragraph introducing yourself (age, grade, hobbies)
    • An <hr> horizontal line to separate sections
    • Another h2 heading that says "My Interests"
    • A paragraph listing your interests, use <br> tags to put each interest on a new line
    • An <hr> to separate sections again
    • An h2 heading that says "My Favorite Website"
    • A link to your favorite website (use <a> tag)
  5. Save and open in your browser!
<html>
<head>
<title>My Profile Page</title>
</head>
<body>
<h1>Alex Johnson</h1>
<h2>About Me</h2>
<p>Hi! I'm Alex, I'm 13 years old and in 8th grade. I love playing basketball and coding!</p>
<hr>
<h2>My Interests</h2>
<p>
Web development<br>
Video games<br>
Science experiments<br>
Reading sci-fi books
</p>
<hr>
<h2>My Favorite Website</h2>
<p>Check out <a href="https://www.nasa.gov">NASA's website</a> - it's amazing!</p>
</body>
</html>

💡 Tips & Reminders
  • Links: The href attribute contains the website address
  • Link text: The text between <a> and </a> is what users click
  • Headings: Use h1 for main title, h2 for sections
  • Paragraphs: Each <p> tag creates a new paragraph with spacing
  • <br> tag: Creates a line break within text (no closing tag needed)
  • <hr> tag: Creates a horizontal divider line (no closing tag needed)
  • Organization: Use proper indentation to make code readable
