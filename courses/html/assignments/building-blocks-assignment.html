🧱 Assignment: Building Blocks with Display Properties

Instructions

Goal: Master block, inline, and inline-block display properties by building visual layouts.

Time: 60-90 minutes

Tools needed: Text editor and web browser

Submission: Create a file named building-blocks.html

📚 Display Properties Reference

Property Behavior Width/Height Margins
display: block; Takes full width, starts new line ✓ Can set ✓ All sides work
display: inline; Flows with text, no line break ✗ Cannot set Only left/right work
display: inline-block; Flows inline but acts like block ✓ Can set ✓ All sides work

📝 Tasks

Task 1: Color Blocks (15 points)

Create 3 colored blocks using display: block;

Requirements:

  • Use <div> elements
  • Add display: block;
  • Each block: different background color, 100px height, padding 20px
  • Add text inside each block

Example Code:

<div style="display: block; background-color: #667eea; 
            height: 100px; padding: 20px; margin: 10px 0; 
            color: white;">
    Block 1
</div>

Expected Result:

Block 1
Block 2
Block 3

Task 2: Inline Tags (15 points)

Create a paragraph with inline highlighted words.

Requirements:

  • Use <p> for paragraph
  • Use <span> with display: inline;
  • Highlight at least 5 words with different colors
  • Add background color and padding to spans

Example Code:

<p style="font-size: 18px;">
    I love <span style="display: inline; background-color: yellow; 
    padding: 3px;">HTML</span> and 
    <span style="display: inline; background-color: lightblue; 
    padding: 3px;">CSS</span>.
</p>

Expected Result:

I love HTML and CSS and JavaScript for web development.

Task 3: Button Row with Inline-Block (20 points)

Create a row of buttons using display: inline-block;

Requirements:

  • Create 4 buttons using <div>
  • Each button: display: inline-block;
  • Width: 150px, Height: 50px
  • Different colors, centered text
  • Margin: 5px between buttons

Example Code:

<div style="display: inline-block; width: 150px; height: 50px;
            background-color: #667eea; color: white; 
            text-align: center; line-height: 50px; 
            margin: 5px; border-radius: 5px;">
    Button 1
</div>

Task 4: Card Grid (25 points)

Create a grid of product cards using display: inline-block;

Requirements:

  • Create 6 cards (2 rows of 3)
  • Each card: width 250px, padding 20px
  • Use display: inline-block;
  • Each card contains: heading, image placeholder, description, price
  • Add border and margin to cards

Example Code:

<div style="display: inline-block; width: 250px; 
            border: 2px solid #ddd; padding: 20px; 
            margin: 10px; border-radius: 8px; 
            vertical-align: top;">
    
    <h3 style="margin-top: 0; color: #667eea;">Product 1</h3>
    
    <div style="width: 100%; height: 150px; 
                background-color: #e9ecef; margin: 10px 0; 
                text-align: center; line-height: 150px; 
                color: #666;">
        Image Placeholder
    </div>
    
    <p>Quality product description here.</p>
    
    <p style="color: #28a745; font-size: 24px; 
              font-weight: bold;">$49.99</p>
    
</div>

<!-- Repeat 5 more times with different products -->

Task 5: Mixed Layout (25 points)

Create a page header combining all three display types.

Requirements:

  • Container div (block) with background color
  • Logo div (inline-block) on left - 100x100px
  • Navigation links (inline-block) on right
  • Badge spans (inline) inside navigation

Structure:

<div style="display: block; background-color: #2c3e50; 
            padding: 20px;">
    
    <!-- Logo -->
    <div style="display: inline-block; width: 100px; 
                height: 100px; background-color: #667eea; 
                color: white; text-align: center; 
                line-height: 100px;">
        LOGO
    </div>
    
    <!-- Nav Links -->
    <div style="display: inline-block; margin-left: 50px;">
        <a href="#" style="display: inline-block; color: white; 
                          padding: 10px 20px; margin: 0 5px;">
            Home <span style="display: inline; background-color: red; 
                              padding: 2px 6px; border-radius: 10px; 
                              font-size: 12px;">New</span>
        </a>
    </div>
    
</div>

Expected Result:

LOGO
Home New Products 5 About

💡 Pro Tips:

  • Use vertical-align: top; on inline-block elements to align them properly
  • Add text-align: center; to parent for centering inline-block children
  • Remember: inline-block elements can have small gaps between them
  • Use browser DevTools to inspect display properties
