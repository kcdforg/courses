PHP If Statements

Learn how to make decisions in your code with conditional statements.

The If Statement

Use the if statement to execute code only if a condition is true.

<?php
    if (condition) {
        // Code to execute if condition is true
    }
?>

Comparison Operators

  • == - Equal to
  • != - Not equal to
  • === - Identical (equal and same type)
  • !== - Not identical
  • < - Less than
  • > - Greater than
  • <= - Less than or equal to
  • >= - Greater than or equal to

The If...Else Statement

<?php
    $age = 20;
    
    if ($age >= 18) {
        echo "You are an adult";
    } else {
        echo "You are a minor";
    }
?>

The If...Else If...Else Statement

<?php
    $score = 75;
    
    if ($score >= 90) {
        echo "Grade A";
    } else if ($score >= 80) {
        echo "Grade B";
    } else if ($score >= 70) {
        echo "Grade C";
    } else {
        echo "Grade F";
    }
?>

Logical Operators

  • and or && - Both conditions must be true
  • or or || - At least one condition must be true
  • xor - Exactly one condition must be true
  • ! - NOT operator (negation)
<?php
    $age = 20;
    $hasLicense = true;
    
    if ($age >= 18 && $hasLicense) {
        echo "You can drive";
    }
?>

Ternary Operator

A shorthand way to write if...else statements:

<?php
    $age = 20;
    $status = ($age >= 18) ? "Adult" : "Minor";
    echo $status; // Outputs "Adult"
?>
