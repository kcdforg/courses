PHP Variables

Learn how to create and work with variables in PHP.

Creating Variables

Variables in PHP start with a dollar sign ($) followed by the variable name.

<?php
    $name = "John";
    $age = 25;
    $price = 9.99;
    echo $name;
?>

Variable Naming Rules

  • Variable names start with a dollar sign ($)
  • Variable names start with a letter or underscore (_), not a number
  • Variable names can only contain alphanumeric characters and underscores (A-z, 0-9, _)
  • Variable names are case-sensitive ($name and $Name are different)

Scope of Variables

PHP has three different variable scopes:

1. Local Scope

Variables declared inside a function have local scope:

<?php
    function myFunction() {
        $x = 5; // Local scope
        echo $x; // Works
    }
    
    myFunction();
    echo $x; // ERROR - not accessible
?>

2. Global Scope

Variables declared outside functions have global scope:

<?php
    $x = 5; // Global scope
    
    function myFunction() {
        global $x; // Access global variable
        echo $x; // Works
    }
    
    myFunction();
?>

3. Static Scope

Static variables retain their value between function calls:

<?php
    function counter() {
        static $count = 0;
        $count++;
        echo $count; // Increments each call
    }
    
    counter(); // Outputs 1
    counter(); // Outputs 2
    counter(); // Outputs 3
?>

Type Juggling

PHP automatically converts variable types as needed:

<?php
    $x = "5"; // String
    $y = 10; // Integer
    echo $x + $y; // Outputs 15 (converted to integer)
?>
