PHP Installation & Setup

Learn how to install and set up PHP on your computer to start developing.

System Requirements

PHP can run on various platforms. Before installation, ensure you have:

  • A web server (Apache, Nginx, IIS)
  • PHP installed on your server
  • A text editor for writing code
  • A web browser to test your applications

Installation Options

Option 1: Using XAMPP (Recommended for Beginners)

XAMPP is the easiest way to install Apache, MySQL, PHP, and Perl on your computer.

Steps:
  1. Download XAMPP from https://www.apachefriends.org
  2. Run the installer and follow the on-screen instructions
  3. Start Apache and MySQL from the XAMPP Control Panel
  4. Place your PHP files in the htdocs folder
  5. Visit http://localhost/filename.php in your browser

Option 2: Using WAMP (Windows)

WAMP stands for Windows, Apache, MySQL, PHP.

Steps:
  1. Download WAMP from http://www.wampserver.com
  2. Install and run WAMP
  3. Access your files through http://localhost

Option 3: Linux/Mac Command Line

# For Ubuntu/Debian
sudo apt-get install php apache2 apache2-mod-php

# For macOS
brew install php apache2

Verify PHP Installation

Create a test file named test.php:

<?php
    phpinfo();
?>

If you see the PHP configuration details, PHP is successfully installed!

Your First PHP Script

Create a file named hello.php:

<?php
    echo "Hello, World!";
?>

Save it in your web server's root directory and visit it in your browser.
