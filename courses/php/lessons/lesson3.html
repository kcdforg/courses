PHP Syntax Basics

Learn the fundamental syntax rules and structure of PHP code.

PHP Tags

PHP code must be enclosed within PHP tags:

<?php
    // PHP code goes here
?>

Statements and Comments

Every PHP statement ends with a semicolon (;). Comments are ignored by the parser.

<?php
    echo "Hello World"; // This is a comment
    /* This is a multi-line
       comment */
    // Another comment
?>

Echo and Print

Both echo and print output text, but echo is slightly faster.

<?php
    echo "This is output using echo";
    print "This is output using print";
    echo "Output", " ", "multiple", " ", "strings";
?>

Whitespace and Line Breaks

Extra whitespace, line breaks, and indentation have no effect on PHP code execution.

<?php
    echo "Hello";
    
    
    echo "World"; // Same output as above
?>

Case Sensitivity

  • Keywords are NOT case-sensitive (echo, print, etc.)
  • User-defined functions are NOT case-sensitive
  • Variable names ARE case-sensitive
Example: 
<?php
    ECHO "Hello"; // Works
    echo "World"; // Works
    $name = "John"; // Correct
    echo $NAME; // ERROR - undefined variable
?>
