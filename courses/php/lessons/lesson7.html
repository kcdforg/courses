PHP Switch Statements

Use switch statements to select one of several code blocks to execute.

Switch Statement Syntax

<?php
    switch (expression) {
        case value1:
            // Code if expression equals value1
            break;
        case value2:
            // Code if expression equals value2
            break;
        default:
            // Code if expression doesn't match any case
    }
?>

How It Works

  • The expression is evaluated once
  • The value is compared with each case
  • When a match is found, the code executes
  • The break statement stops execution
  • The default keyword handles no matches

Example

<?php
    $day = 3;
    
    switch ($day) {
        case 1:
            echo "Monday";
            break;
        case 2:
            echo "Tuesday";
            break;
        case 3:
            echo "Wednesday";
            break;
        case 4:
            echo "Thursday";
            break;
        case 5:
            echo "Friday";
            break;
        default:
            echo "Weekend";
    }
    // Outputs "Wednesday"
?>

The Default Keyword

The default keyword specifies code to run if no cases match:

<?php
    $color = "purple";
    
    switch ($color) {
        case "red":
            echo "Red";
            break;
        case "blue":
            echo "Blue";
            break;
        default:
            echo "Unknown color";
    }
    // Outputs "Unknown color"
?>

Important Notes

  • Always include break; statements to avoid fall-through behavior
  • Use strict comparison (===) in some cases
  • Switch is more efficient than multiple if...else for many conditions
