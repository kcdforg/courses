PHP Loops

Learn how to repeat code blocks using different loop types.

While Loop

A while loop executes code repeatedly while a condition is true:

<?php
    $i = 0;
    while ($i < 5) {
        echo "Number: " . $i . "<br>";
        $i++;
    }
?>

Do...While Loop

Executes code at least once, then repeats while condition is true:

<?php
    $i = 0;
    do {
        echo "Number: " . $i . "<br>";
        $i++;
    } while ($i < 5);
?>

For Loop

Used when you know exactly how many times to repeat:

<?php
    for ($i = 0; $i < 5; $i++) {
        echo "Number: " . $i . "<br>";
    }
?>

ForEach Loop

Used to loop through arrays:

<?php
    $fruits = array("Apple", "Banana", "Orange");
    
    foreach ($fruits as $fruit) {
        echo $fruit . "<br>";
    }
    
    // With associative arrays
    $person = array("name" => "John", "age" => 25);
    foreach ($person as $key => $value) {
        echo $key . ": " . $value . "<br>";
    }
?>

Break and Continue

break exits the loop, continue skips current iteration:

<?php
    for ($i = 0; $i < 10; $i++) {
        if ($i == 3) {
            continue; // Skip 3
        }
        if ($i == 7) {
            break; // Exit loop
        }
        echo $i . "<br>";
    }
?>

Nested Loops

<?php
    for ($i = 1; $i <= 3; $i++) {
        for ($j = 1; $j <= 3; $j++) {
            echo $i . "-" . $j . " ";
        }
        echo "<br>";
    }
?>
