PHP Functions

Learn how to create and use reusable functions in PHP.

Creating Functions

A function is a reusable block of code:

<?php
    function sayHello() {
        echo "Hello, World!";
    }
    
    sayHello(); // Call the function
?>

Function Parameters

Functions can accept parameters:

<?php
    function greet($name) {
        echo "Hello, " . $name . "!";
    }
    
    greet("John"); // Outputs "Hello, John!"
    greet("Jane");
?>

Multiple Parameters

<?php
    function add($a, $b) {
        return $a + $b;
    }
    
    echo add(5, 3); // Outputs 8
?>

Default Parameter Values

Provide default values for parameters:

<?php
    function welcome($name = "Guest") {
        echo "Welcome, " . $name . "!";
    }
    
    welcome(); // Outputs "Welcome, Guest!"
    welcome("John"); // Outputs "Welcome, John!"
?>

Return Values

Use return to get a value from a function:

<?php
    function multiply($a, $b) {
        return $a * $b;
    }
    
    $result = multiply(4, 5);
    echo $result; // Outputs 20
?>

Variable Scope in Functions

Variables inside functions are local by default:

<?php
    $globalVar = "I'm global";
    
    function test() {
        global $globalVar; // Access global variable
        echo $globalVar;
    }
    
    test();
?>

Type Declarations

Specify parameter and return types:

<?php
    function calculate(int $a, int $b): int {
        return $a + $b;
    }
    
    echo calculate(5, 3); // Returns 8
?>
