PHP Data Types

Understand the different data types available in PHP.

PHP Data Types

PHP supports the following data types:

1. String

A string is a sequence of characters enclosed in quotes.

<?php
    $str1 = "Hello World"; // Double quotes
    $str2 = 'Hello World'; // Single quotes
    echo $str1;
?>

2. Integer

An integer is a whole number without decimals.

<?php
    $int = 1234;
    $int = -1234; // Negative
    $int = 0x8C; // Hexadecimal
    var_dump($int);
?>

3. Float (Double)

A float is a number with decimal points.

<?php
    $float = 9.99;
    $float = 2.0e3; // Scientific notation
    var_dump($float);
?>

4. Boolean

A boolean is either TRUE or FALSE.

<?php
    $bool1 = true;
    $bool2 = false;
    
    if ($bool1) {
        echo "True!";
    }
?>

5. Array

An array stores multiple values in a single variable.

<?php
    $fruits = array("Apple", "Banana", "Orange");
    echo $fruits[0]; // Outputs "Apple"
    
    // Short syntax
    $colors = ["Red", "Green", "Blue"];
?>

6. Object

An object is an instance of a class.

<?php
    class Car {
        public $color = "red";
    }
    
    $car = new Car();
    echo $car->color;
?>

7. NULL

A variable with NULL value represents an empty value.

<?php
    $x = "Hello";
    $x = null; // Now empty
    var_dump($x);
?>

Type Checking Functions

  • is_string() - Check if variable is a string
  • is_int() - Check if variable is an integer
  • is_float() - Check if variable is a float
  • is_bool() - Check if variable is a boolean
  • is_array() - Check if variable is an array
  • is_null() - Check if variable is NULL
