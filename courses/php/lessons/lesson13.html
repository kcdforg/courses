PHP File Handling

Learn how to work with files using PHP.

file_exists() - Check if File Exists

<?php
    if (file_exists("test.txt")) {
        echo "File exists";
    } else {
        echo "File does not exist";
    }
?>

Reading Files

file_get_contents() - Read Entire File

<?php
    $content = file_get_contents("test.txt");
    echo $content;
?>

file() - Read into Array

<?php
    $lines = file("test.txt");
    
    foreach ($lines as $line) {
        echo $line . "<br>";
    }
?>

readfile() - Output File

<?php
    readfile("test.txt"); // Directly outputs file content
?>

Writing Files

file_put_contents() - Write to File

<?php
    $text = "Hello, World!";
    
    // Create or overwrite
    file_put_contents("test.txt", $text);
    
    // Append to file
    file_put_contents("test.txt", $text, FILE_APPEND);
?>

fopen() / fwrite() / fclose()

<?php
    // Open file
    $file = fopen("test.txt", "w");
    
    // Write to file
    fwrite($file, "Hello World");
    
    // Close file
    fclose($file);
?>

File Modes

  • "r" - Read only
  • "w" - Write (creates or truncates)
  • "a" - Append (creates if not exists)
  • "x" - Create new file (error if exists)
  • "r+" - Read and write
  • "w+" - Write and read (truncates)
  • "a+" - Append and read

Additional File Functions

unlink() - Delete File

<?php
    if (file_exists("test.txt")) {
        unlink("test.txt");
        echo "File deleted";
    }
?>

copy() - Copy File

<?php
    copy("test.txt", "test_copy.txt");
?>

rename() - Rename File

<?php
    rename("test.txt", "myfile.txt");
?>

is_file() / is_dir()

<?php
    if (is_file("test.txt")) {
        echo "It's a file";
    }
    
    if (is_dir("mydir")) {
        echo "It's a directory";
    }
?>

filesize() - Get File Size

<?php
    echo filesize("test.txt"); // Returns size in bytes
?>
