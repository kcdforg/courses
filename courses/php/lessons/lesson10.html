PHP Arrays

Learn how to create and work with arrays in PHP.

Indexed Arrays

Arrays with numeric indices:

<?php
    $fruits = array("Apple", "Banana", "Orange");
    echo $fruits[0]; // Outputs "Apple"
    
    // Short syntax
    $colors = ["Red", "Green", "Blue"];
    echo $colors[1]; // Outputs "Green"
?>

Associative Arrays

Arrays with named keys:

<?php
    $person = array(
        "name" => "John",
        "age" => 25,
        "city" => "New York"
    );
    
    echo $person["name"]; // Outputs "John"
    echo $person["age"]; // Outputs 25
?>

Adding Elements

<?php
    $fruits = ["Apple", "Banana"];
    
    // Add element
    $fruits[] = "Orange"; // Index 2
    $fruits[3] = "Mango";
    
    // For associative arrays
    $person = ["name" => "John"];
    $person["age"] = 25;
?>

Removing Elements

<?php
    $fruits = ["Apple", "Banana", "Orange"];
    
    unset($fruits[1]); // Removes "Banana"
?>

Getting Array Length

<?php
    $fruits = ["Apple", "Banana", "Orange"];
    echo count($fruits); // Outputs 3
    echo sizeof($fruits); // Also outputs 3
?>

Multidimensional Arrays

Arrays containing other arrays:

<?php
    $matrix = array(
        array(1, 2, 3),
        array(4, 5, 6),
        array(7, 8, 9)
    );
    
    echo $matrix[0][0]; // Outputs 1
    echo $matrix[1][2]; // Outputs 6
    echo $matrix[2][1]; // Outputs 8
?>

Looping Through Arrays

<?php
    $fruits = ["Apple", "Banana", "Orange"];
    
    // Using foreach
    foreach ($fruits as $fruit) {
        echo $fruit . " ";
    }
    
    // Using for loop
    for ($i = 0; $i < count($fruits); $i++) {
        echo $fruits[$i] . " ";
    }
?>
