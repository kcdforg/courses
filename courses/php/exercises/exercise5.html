Exercise 5: Number Classification

Classify numbers based on various properties.

Objective

Use multiple if-else statements to classify a number.

Requirements

  • Check if number is positive, negative, or zero
  • Check if number is even or odd
  • Combine both checks in output
  • Test with multiple numbers

Starter Code

<?php
    $number = 42;
    
    // TODO: Check if positive/negative/zero
    // TODO: Check if even/odd
    // TODO: Display classification
    
?>

Expected Output

Number: 42
Classification: Positive, Even

Number: -15
Classification: Negative, Odd

Number: 0
Classification: Zero

Hint

Use the modulo operator (%) to check even/odd: $number % 2 == 0 for even.
