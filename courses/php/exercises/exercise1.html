Exercise 1: Variable Declaration

Create variables for a person and display them in a formatted output.

Objective

Learn how to create and work with variables of different data types.

Requirements

  • Create variables for person's name, age, email, and salary
  • Assign appropriate values to each variable
  • Display the information in a nicely formatted way
  • Use string concatenation or interpolation

<?php
    // TODO: Create variables for:

    // - Age (integer)
    // - Email (string)
    // - Salary (float)
    
    // TODO: Display the information
    
?>

Expected Output

Person Information:
Name: John Doe
Age: 25
Email: john@example.com
Salary: $50000.00

Hints

  • Use the concatenation operator (.) to combine strings
  • Use number_format() for formatting salary
  • Use echo to display output
  • Use <br> for line breaks in HTML output

Challenge Extension

  • Add more properties like city, department, hire date
  • Calculate and display years of service
  • Use CSS styling for better presentation
