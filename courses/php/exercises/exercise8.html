Exercise 8: Times Table

Generate and display multiplication table for a given number.

Objective

Practice using loops to generate repetitive output.

Requirements

  • Take a number as input (e.g., 5)
  • Generate multiplication table from 1 to 10
  • Format the output nicely
  • Display with HTML formatting

Starter Code

<?php
    $number = 5;
    
    echo "<h3>Multiplication Table of " . $number . "</h3>";
    
    // TODO: Create loop to generate table
    
?>

Expected Output

Multiplication Table of 5
5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
5 x 4 = 20
5 x 5 = 25
5 x 6 = 30
5 x 7 = 35
5 x 8 = 40
5 x 9 = 45
5 x 10 = 50

Challenge

Display multiplication tables for multiple numbers (1-10) using nested loops.
