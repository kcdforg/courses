Exercise 3: Type Conversion

Convert values between different data types.

Objective

Learn how PHP handles type conversions and use type checking functions.

Requirements

  • Convert string "100" to integer and perform arithmetic
  • Convert integer 50 to string and concatenate
  • Convert "3.14" to float and use in calculation
  • Use is_string(), is_int(), is_float() functions
  • Convert to boolean and test values

Original: "100" (string)

Useful Functions

  • (string) - Cast to string

  • is_string(), is_int(), is_float()
  • gettype() - Get variable type
