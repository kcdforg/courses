Exercise 6: Day Finder

Convert day numbers to day names using switch statements.

Objective

Practice using switch statements to handle multiple cases.

Requirements

  • Take a day number (1-7)
  • Convert to day name (Monday-Sunday)
  • Display whether it's a weekday or weekend
  • Add description for each day

Day Mapping

  • 1 = Monday (Weekday)
  • 2 = Tuesday (Weekday)
  • 3 = Wednesday (Weekday)
  • 4 = Thursday (Weekday)
  • 5 = Friday (Weekday)
  • 6 = Saturday (Weekend)
  • 7 = Sunday (Weekend)

Starter Code

<?php
    $day = 3;
    
    // TODO: Use switch to find day name
    // TODO: Determine weekday/weekend
    // TODO: Display information
    
?>

Expected Output

Day Number: 3
Day Name: Wednesday
Type: Weekday
Description: Mid-week, keep working!
