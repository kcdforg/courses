Exercise 4: Grade Calculator

Assign grades based on student marks using if-else statements.

Objective

Use conditional statements to calculate grades based on marks.

Requirements

  • Input a student's marks (0-100)
  • Assign grades: A (90+), B (80-89), C (70-79), D (60-69), F (below 60)
  • Display the marks and corresponding grade
  • Add pass/fail status

Grading Scale

  • A: 90-100 (Excellent)
  • B: 80-89 (Good)
  • C: 70-79 (Average)
  • D: 60-69 (Below Average)
  • F: Below 60 (Fail)
