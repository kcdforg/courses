Exercise 14: Temperature Converter

Write functions to convert between temperature scales.

Objective

Create functions for temperature conversions using formulas.

Formulas

  • Celsius to Fahrenheit: (C × 9/5) + 32
  • Fahrenheit to Celsius: (F - 32) × 5/9
  • Celsius to Kelvin: C + 273.15
  • Kelvin to Celsius: K - 273.15

Requirements

  • Create conversion functions for all three scales
  • Input: temperature value
  • Output: converted value with rounded to 2 decimals
  • Test with multiple values
