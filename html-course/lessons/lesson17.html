Lesson 17: Final Project - Build Your Portfolio Website

Project Overview

Congratulations on completing all 16 lessons! Now it's time to put everything together and create your own personal portfolio website.

🎯 Project Goal

Create a multi-page personal portfolio website that showcases what you've learned and tells your story.

Required Pages

1. Home Page (index.html)

Must Include:
  • Header with your name and navigation menu
  • Welcome section with H1 heading
  • Brief introduction about yourself
  • Image of yourself or avatar
  • Links to other pages
  • Footer with copyright and contact info

2. About Page (about.html)

Must Include:
  • Detailed bio with multiple paragraphs
  • Your hobbies (unordered list)
  • Your skills (ordered list)
  • Educational background
  • Personal interests

3. Projects/Gallery Page (projects.html)

Must Include:
  • At least 3 project showcases
  • Images for each project
  • Description of each project
  • Use sections or articles for each project

4. Contact Page (contact.html)

Must Include:
  • Contact form with name, email, and message fields
  • Radio buttons for reason (feedback, question, collaboration)
  • Checkbox for newsletter subscription
  • Submit button
  • Your social media links (if any)

Technical Requirements

✓ HTML Structure Checklist

  • ✓ Valid HTML5 structure (DOCTYPE, html, head, body)
  • ✓ Proper meta tags (charset, viewport, description)
  • ✓ Meaningful page titles for each page
  • ✓ Semantic HTML (header, nav, main, section, article, footer)
  • ✓ Consistent navigation on all pages

✓ Content Requirements

  • ✓ At least 10 different HTML tags used
  • ✓ Proper heading hierarchy (H1 → H2 → H3)
  • ✓ At least 5 images with alt text
  • ✓ At least 1 table
  • ✓ Multiple lists (ordered and unordered)
  • ✓ At least 10 internal and external links
  • ✓ A complete contact form

✓ Best Practices

  • ✓ All images have descriptive alt text
  • ✓ All form inputs have labels
  • ✓ Use HTML entities for special characters
  • ✓ Descriptive link text (no "click here")
  • ✓ Code is properly indented and organized
  • ✓ Valid HTML (no unclosed tags)

Bonus Features (Optional)

Extra Credit:
  • Add a table showing your weekly schedule or skills
  • Embed a YouTube video on your projects page
  • Add an iframe with Google Maps to your location
  • Include audio file (favorite song or introduction)
  • Create a nested list for complex information
  • Add Open Graph meta tags for social sharing
  • Use div and span for additional styling hooks
  • Include HTML entities and special symbols

Tips for Success

  • Start Simple: Get basic structure working first, then add details
  • Test Often: Check your pages in a browser frequently
  • Validate: Use W3C HTML Validator to check for errors
  • Be Creative: Make it personal and unique to you
  • Review Lessons: Go back to lessons when you forget syntax
  • Ask for Help: Don't hesitate to review documentation
