🔍 Lesson 14: Meta Tags & SEO Basics

Learn how to use meta tags to improve your website's search engine visibility and control how your pages appear in search results and social media!

What are Meta Tags?

Meta tags provide information about your HTML document to browsers and search engines. They go in the <head> section and aren't visible on the page, but they're crucial for SEO (Search Engine Optimization).

Essential Meta Tags:

<meta charset="UTF-8">

Defines character encoding (always include this!)

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Makes your site mobile-responsive

<meta name="description" content="...">

Appears in search engine results (150-160 characters)

<meta name="author" content="...">

Identifies the page author

🎯 SEO Best Practices

  • Use ONE <h1> per page (main title)
  • Write descriptive <title> tags (50-60 characters)
  • Create unique meta descriptions (150-160 characters)
  • Use semantic HTML tags for better structure
  • Add alt text to ALL images
  • Create descriptive, readable URLs
  • Use internal links between pages
  • Make your site mobile-friendly

🎯 Progressive Activities

Complete these activities to master meta tags and SEO!

Activity 1

Add Essential Meta Tags

Beginner

Learn to add the most important meta tags to your HTML document.

Instructions:

  1. In the <head> section, add charset meta tag
  2. Add viewport meta tag for mobile responsiveness
  3. Add a descriptive title tag
  4. Add author meta tag with your name
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="author" content="Your Name">
    <title>My Awesome Website - Home Page</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>Content goes here...</p>
</body>
</html>

Expected Output:

The page won't look different, but search engines and browsers can now properly understand your page.

💡 Hints:

  • UTF-8 supports all languages and special characters
  • Viewport meta tag is ESSENTIAL for mobile devices
  • Title appears in browser tab and search results
Activity 2

Add Meta Description

Beginner

Create a meta description that will appear in search engine results.

Instructions:

  1. Add a meta description tag
  2. Write a clear, compelling description (150-160 characters)
  3. Include relevant keywords naturally
  4. Make it descriptive and enticing for users
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Learn HTML basics from scratch. 
          Step-by-step tutorials for beginners covering tags, forms, 
          tables, and more. Start building websites today!">
    <title>HTML Tutorial for Beginners</title>
</head>
<body>
    <h1>HTML Tutorial</h1>
</body>
</html>

Expected Output:

This description will appear in Google search results below your page title.

💡 Hints:

  • Keep it between 150-160 characters
  • Include your main keyword naturally
  • Write for humans, not just search engines
  • Make it actionable and compelling
Activity 3

Create SEO-Friendly Content Structure

Intermediate

Build a page with proper heading hierarchy for better SEO.

Instructions:

  1. Use ONE <h1> for the main page title
  2. Use <h2> for major sections
  3. Use <h3> for subsections
  4. Maintain logical hierarchy (don't skip levels)
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Complete guide to HTML basics">
    <title>HTML Basics Guide</title>
</head>
<body>
    <h1>HTML Basics Guide</h1>
    <p>Learn the fundamentals of HTML.</p>
    
    <h2>Getting Started</h2>
    <p>Introduction to HTML.</p>
    
    <h3>What is HTML?</h3>
    <p>HTML stands for HyperText Markup Language.</p>
    
    <h3>Why Learn HTML?</h3>
    <p>HTML is the foundation of all websites.</p>
    
    <h2>Basic Tags</h2>
    <p>Learn essential HTML tags.</p>
    
    <h3>Headings</h3>
    <p>Use h1 through h6 for headings.</p>
    
    <h3>Paragraphs</h3>
    <p>Use the p tag for text paragraphs.</p>
</body>
</html>

Expected Output:

A well-structured page with clear heading hierarchy that search engines can easily understand.

💡 Hints:

  • Only use ONE h1 per page
  • Don't skip heading levels (h1 → h3 without h2)
  • Headings should be descriptive and contain keywords
  • Use headings for structure, not just styling
Activity 4

Add Social Media Tags

Advanced

Add Open Graph meta tags to control how your page appears when shared on social media.

Instructions:

  1. Add all essential meta tags
  2. Add Open Graph (og:) tags for social sharing
  3. Include og:title, og:description, og:image, og:url
  4. Add og:type (usually "website")
<html lang="en">
<head>
    <!-- Essential Meta Tags -->
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Learn web development from scratch with 
          our comprehensive HTML tutorial">
    <meta name="author" content="Web Dev Academy">
    
    <!-- Open Graph Meta Tags (Social Media) -->
    <meta property="og:title" content="HTML Tutorial for Beginners">
    <meta property="og:description" content="Master HTML with our 
          step-by-step guide. Perfect for beginners!">
    <meta property="og:image" content="https://example.com/thumbnail.jpg">
    <meta property="og:url" content="https://example.com/html-tutorial">
    <meta property="og:type" content="website">
    
    <title>HTML Tutorial for Beginners - Web Dev Academy</title>
</head>
<body>
    <h1>HTML Tutorial for Beginners</h1>
    <p>Start your web development journey today!</p>
</body>
</html>

Expected Output:

When shared on Facebook, LinkedIn, or Twitter, your page will display with the custom title, description, and image you specified.

🏆 Challenge:

  • Add Twitter Card meta tags (twitter:card, twitter:title, etc.)
  • Test your Open Graph tags using Facebook's Sharing Debugger
  • Add a favicon link tag
