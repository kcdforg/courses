📦 Lesson 12: Div & Span Elements

Learn about the two most commonly used container elements in HTML - <div> for block-level grouping and <span> for inline styling!

What are Div and Span?

<div> and <span> are generic container elements used to group content for styling or scripting purposes. They don't have any semantic meaning but are essential for layout and styling.

The Two Container Elements:

<div>...</div>

Block-level element: Takes full width, starts on a new line

<span>...</span>

Inline element: Only takes needed width, stays in line with text

🎯 Key Differences

Feature <div> (Block) <span> (Inline)
Display Takes full width available Takes only needed width
Line Break Starts on new line Stays in same line
Width/Height Can set width & height Cannot set width & height
Use Case Layout sections, grouping blocks Styling parts of text

🎯 Progressive Activities

Complete these activities to master div and span elements!

Activity 1

Create Your First Div

Beginner

Learn to use the <div> element to group content together.

Instructions:

  1. Create a <div> with inline style for background color
  2. Add a heading inside the div
  3. Add a paragraph inside the div
  4. Notice how the div takes full width
<html>
<body>

<div style="background-color: lightblue; padding: 20px;">
    <h2>Welcome Section</h2>
    <p>This content is grouped inside a div element.</p>
</div>

<p>This paragraph is outside the div.</p>

</body>
</html>

Expected Output:

A blue box containing a heading and paragraph, followed by a regular paragraph outside the box.

💡 Hints:

  • <div> is a block-level element
  • It automatically starts on a new line
  • It takes up the full width available
  • Use padding to add space inside the div
Activity 2

Style Text with Span

Beginner

Learn to use the <span> element to style parts of text.

Instructions:

  1. Create a paragraph with some text
  2. Wrap one word in a <span> tag
  3. Apply inline style to make the span text red
  4. Notice how span stays inline with the text
<html>
<body>

<p>This is <span style="color: red; font-weight: bold;">important</span> text.</p>

<p>HTML is <span style="color: green;">easy</span> to learn!</p>

</body>
</html>

Expected Output:

Paragraphs where specific words are highlighted in color without breaking the text flow.

💡 Hints:

  • <span> is an inline element
  • It doesn't break the text flow or start a new line
  • Perfect for styling parts of text within a paragraph
Activity 3

Compare Block vs Inline

Intermediate

Create an example that clearly shows the difference between block and inline elements.

Instructions:

  1. Create 3 <div> elements with different background colors
  2. Create a paragraph with 3 <span> elements with different colors
  3. Observe how divs stack vertically
  4. Observe how spans stay in the same line
<html>
<body>

<h3>Block-level (div):</h3>
<div style="background-color: lightblue; padding: 10px;">Div 1</div>
<div style="background-color: lightgreen; padding: 10px;">Div 2</div>
<div style="background-color: lightyellow; padding: 10px;">Div 3</div>

<h3>Inline (span):</h3>
<p>
    <span style="background-color: lightblue; padding: 5px;">Span 1</span>
    <span style="background-color: lightgreen; padding: 5px;">Span 2</span>
    <span style="background-color: lightyellow; padding: 5px;">Span 3</span>
</p>

</body>
</html>

Expected Output:

Three divs stacked vertically, each taking full width. Three spans side-by-side in the same line.

💡 Hints:

  • Block elements stack vertically (one per line)
  • Inline elements flow horizontally (multiple per line)
  • This is the fundamental difference between div and span
Activity 4

Use Classes with Div

Intermediate

Learn to use the class attribute with divs for better organization.

Instructions:

  1. Create 3 divs with class="box"
  2. Add inline styles to each div for border and padding
  3. Add different content to each box
  4. Notice how classes help organize similar elements
<html>
<body>

<div class="box" style="border: 2px solid blue; padding: 15px; margin: 10px 0;">
    <h3>Box 1</h3>
    <p>First box content</p>
</div>

<div class="box" style="border: 2px solid blue; padding: 15px; margin: 10px 0;">
    <h3>Box 2</h3>
    <p>Second box content</p>
</div>

<div class="box" style="border: 2px solid blue; padding: 15px; margin: 10px 0;">
    <h3>Box 3</h3>
    <p>Third box content</p>
</div>

</body>
</html>

Expected Output:

Three bordered boxes stacked vertically, each containing a heading and paragraph.

💡 Hints:

  • Classes help group similar elements
  • You can apply the same class to multiple elements
  • In real projects, you'd use CSS files instead of inline styles
Activity 5

Create Nested Divs

Intermediate

Learn to nest divs inside each other for complex layouts.

Instructions:

  1. Create an outer div with class "container"
  2. Inside, create 3 inner divs for header, content, and footer
  3. Style each div with different background colors
  4. Add content to each section
<html>
<body>

<div class="container" style="border: 3px solid black; padding: 10px;">
    
    <div class="header" style="background-color: #667eea; color: white; 
                                 padding: 15px;">
        <h2>Website Header</h2>
    </div>
    
    <div class="content" style="background-color: #f8f9fa; 
                                  padding: 15px; margin: 10px 0;">
        <p>Main content goes here</p>
    </div>
    
    <div class="footer" style="background-color: #2c3e50; color: white; 
                                 padding: 15px;">
        <p>Website Footer</p>
    </div>
    
</div>

</body>
</html>

Expected Output:

A container box with three sections: blue header, gray content area, and dark footer.

💡 Hints:

  • Nesting divs is common for creating layouts
  • The outer div acts as a container for inner sections
  • Use meaningful class names (header, content, footer)
Activity 6

Build a Product Card

Advanced

Create a complete product card using both divs for layout and spans for inline styling.

Instructions:

  1. Create an outer div for the product card with border and padding
  2. Add an inner div for product title
  3. Add a div for product description
  4. Add a div for price, using <span> to style the price number
  5. Add a "Sale" badge using styled <span>
<html>
<body>

<div class="product-card" style="border: 2px solid #ddd; padding: 20px; 
                                  max-width: 300px; border-radius: 8px;">
    
    <div class="product-header">
        <h3>Wireless Headphones</h3>
        <span style="background-color: red; color: white; padding: 3px 8px; 
                     border-radius: 3px; font-size: 12px;">SALE</span>
    </div>
    
    <div class="product-description" style="margin: 15px 0;">
        <p>High-quality wireless headphones with noise cancellation 
           and 20-hour battery life.</p>
    </div>
    
    <div class="product-price">
        <p>Price: 
           <span style="font-size: 24px; color: green; 
                        font-weight: bold;">$79.99</span>
        </p>
        <p>Was: 
           <span style="text-decoration: line-through; 
                        color: gray;">$129.99</span>
        </p>
    </div>
    
    <div class="product-button">
        <button style="background-color: #667eea; color: white; 
                       padding: 10px 20px; border: none; 
                       border-radius: 5px; cursor: pointer;">
            Add to Cart
        </button>
    </div>
    
</div>

</body>
</html>

Expected Output:

A professional-looking product card with title, sale badge, description, original/sale prices, and an "Add to Cart" button.

🏆 Challenge:

  • Create 2-3 product cards side by side
  • Add a product image placeholder div
  • Add a "Best Seller" badge to one card
  • Try using semantic tags where appropriate (consider what we learned in Lesson 11!)

📚 When to Use What

Use Semantic Tags First:

Always prefer semantic HTML when available:

  • <header>, <footer>, <nav>, <article>, <section>

Use <div> for:

  • Generic containers without semantic meaning
  • Layout wrappers and grouping
  • Creating grid or flexbox layouts
  • Styling purposes when no semantic tag fits

Use <span> for:

  • Styling parts of text within paragraphs
  • Inline elements that need special styling
  • Small inline containers (icons, badges, labels)
