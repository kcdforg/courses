🔤 Lesson 16: HTML Entities & Special Characters

Learn to display special characters, symbols, and reserved HTML characters using HTML entities. Essential for showing <, >, ©, and more!

What are HTML Entities?

HTML entities are special codes used to display characters that are either reserved in HTML (like < and >) or not available on your keyboard (like © and €). They start with & and end with ;

🎯 Why Use Entities?

  • Display reserved HTML characters (<, >, &)
  • Show special symbols (©, ®, €, ™)
  • Add mathematical symbols (×, ÷, ≠)
  • Include accented characters (é, ñ, ü)
  • Add arrows and other symbols (→, ←, ★)

Entity Syntax:

&entity_name;

Named entity (e.g., &copy; displays ©)

&#number;

Numeric entity (e.g., &#169; displays ©)

📋 Most Common Entities:

Character Entity Name Description
< &lt; Less than
> &gt; Greater than
& &amp; Ampersand
© &copy; Copyright
® &reg; Registered trademark
€ &euro; Euro currency

🎯 Progressive Activities

Complete these activities to master HTML entities!

Activity 1

Display Reserved Characters

Beginner

Learn to display HTML code on your page using entities.

Instructions:

  1. Create a paragraph showing HTML code example
  2. Use &lt; for <
  3. Use &gt; for >
  4. The HTML code should appear as text, not be interpreted
<html>
<body>

<h2>HTML Code Example</h2>

<p>To create a paragraph, use: &lt;p&gt;Your text&lt;/p&gt;</p>

<p>To create a heading, use: &lt;h1&gt;Title&lt;/h1&gt;</p>

<p>To add a link, use: &lt;a href="url"&gt;Link text&lt;/a&gt;</p>

</body>
</html>

Expected Output:

The HTML tags appear as visible text instead of being interpreted as actual HTML.

💡 Hints:

  • &lt; displays as <
  • &gt; displays as >
  • Without entities, <p> would create an actual paragraph
Activity 2

Add Copyright and Trademark Symbols

Beginner

Use entities to add professional copyright and trademark symbols.

Instructions:

  1. Add copyright symbol using &copy;
  2. Add registered trademark using &reg;
  3. Add trademark symbol using &trade;
  4. Create a professional footer with these symbols
<html>
<body>

<h1>Welcome to TechCorp&reg;</h1>

<p>Our flagship product is WebBuilder&trade;, the easiest 
   way to create websites.</p>

<footer>
    <p>&copy; 2025 TechCorp&reg;. All rights reserved.</p>
    <p>WebBuilder&trade; is a trademark of TechCorp.</p>
</footer>

</body>
</html>

Expected Output:

Professional-looking text with ©, ®, and ™ symbols appearing correctly.

💡 Hints:

  • &copy; → ©
  • &reg; → ®
  • &trade; → ™
  • These symbols make your site look professional

    </tr>

A price table with proper currency symbols: €, £, and ¥

  • &euro; → €

Activity 4

Use Mathematical Symbols

Intermediate

Display mathematical equations using HTML entities.

Instructions:

  1. Create mathematical equations
  2. Use &times; for ×
  3. Use &divide; for ÷
  4. Use &plusmn; for ±
  5. Use &ne; for ≠
<html>
<body>

<h2>Math Problems</h2>

<p>Problem 1: 5 &times; 3 = 15</p>

<p>Problem 2: 20 &divide; 4 = 5</p>

<p>Problem 3: &plusmn;10 (plus or minus 10)</p>

<p>Problem 4: 5 &ne; 3 (5 is not equal to 3)</p>

<p>Problem 5: x &le; 10 (x is less than or equal to 10)</p>

<p>Problem 6: y &ge; 5 (y is greater than or equal to 5)</p>

</body>
</html>

Expected Output:

Mathematical equations with proper symbols: ×, ÷, ±, ≠, ≤, ≥

💡 Hints:

  • &times; → ×
  • &divide; → ÷
  • &plusmn; → ±
  • &ne; → ≠
  • &le; → ≤
  • &ge; → ≥
Activity 5

Add Arrows and Special Symbols

Intermediate

Use arrow entities and other special symbols in your content.

Instructions:

  1. Create navigation or instructions using arrows
  2. Use &rarr; for →
  3. Use &larr; for ←
  4. Use &uarr; for ↑
  5. Use &darr; for ↓
<html>
<body>

<h2>Website Navigation</h2>

<p>&larr; Previous Page | Next Page &rarr;</p>

<h3>Instructions</h3>

<p>Step 1: Click "Start" button &darr;</p>
<button>Start</button>

<p>Step 2: Scroll &uarr; to the top when finished</p>

<h3>Flow Chart</h3>

<p>Start &rarr; Process &rarr; End</p>

</body>
</html>

Expected Output:

Text with arrows: ←, →, ↑, ↓ making navigation and instructions clearer.

💡 Hints:

  • &rarr; → → (right arrow)
  • &larr; → ← (left arrow)
  • &uarr; → ↑ (up arrow)
  • &darr; → ↓ (down arrow)
