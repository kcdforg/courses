📚 ID and Class You Will Use

🆔 IDs (Unique - used only once):
id="profile-card" The main card container
id="profile-banner" The colored banner at top
id="profile-avatar" Your profile picture/emoji
id="profile-info" The info section below avatar
📦 Classes (Reusable - can use many times):
class="profile-name" Your name heading
class="profile-title" Your role/title
class="profile-bio" About you paragraph
class="stat-item" Each stat box (Projects, Followers, etc.)
class="skill-tag" Each skill badge
class="social-link" Each social media button

📝 Step-by-Step Instructions

1 Create the HTML File

Create a new file named my-profile.html and add this basic structure:

<!DOCTYPE html>
<html>
<head>
<title>My Profile</title>
<style>
/* We will add CSS here */
</style>
</head>
<body>
<!-- Profile card goes here -->
</body>
</html>

2 Add the Page Background (CSS)

Inside <style>, add this CSS to make a nice dark background:

body {
margin: 0;
padding: 40px;
min-height: 100vh;
display: flex;
justify-content: center;
align-items: center;
background: linear-gradient(135deg, #1a1a2e, #16213e);
font-family: 'Segoe UI', sans-serif;
}

3 Create the Profile Card Container ID

Add the main card div inside <body>:

<!-- HTML -->
<div id="profile-card">
<!-- Everything goes inside here -->
</div>
/* CSS - Add to style section */
#profile-card {
background: white;
width: 350px;
border-radius: 15px;
overflow: hidden;
box-shadow: 0 10px 30px rgba(0,0,0,0.3);
}

4 Add the Banner ID

Add a colorful banner at the top (inside profile-card):

<!-- HTML - inside profile-card -->
<div id="profile-banner"></div>
/* CSS */
#profile-banner {
height: 100px;
background: linear-gradient(135deg, #667eea, #764ba2);
}

5 Add the Avatar ID

Add a circular avatar that overlaps the banner:

<!-- HTML - after banner -->
<div id="profile-avatar">👨‍💻</div>
/* CSS */
#profile-avatar {
width: 100px;
height: 100px;
background: linear-gradient(135deg, #11998e, #38ef7d);
border-radius: 50%;
border: 4px solid white;
margin: -50px auto 15px;
display: flex;
align-items: center;
justify-content: center;
font-size: 40px;
}

6 Add Info Section with Name, Title, Bio ID CLASS

Add the info container with your details:

<!-- HTML - after avatar -->
<div id="profile-info">
<h2 class="profile-name">Your Name</h2>
<p class="profile-title">Student / Developer</p>
<p class="profile-bio">Write about yourself here!</p>
</div>
/* CSS */
#profile-info { padding: 0 25px 25px; text-align: center; }
.profile-name { color: #333; margin: 0 0 5px; }
.profile-title { color: #666; margin: 0 0 15px; }
.profile-bio { color: #555; line-height: 1.5; }

7 Add Stats Section CLASS

Add stats like Projects, Followers, Following (inside profile-info):

<!-- HTML - inside profile-info, after bio -->
<div class="stats-container">
<div class="stat-item">
<div class="stat-number">12</div>
<div class="stat-label">Projects</div>
</div>
<!-- Repeat for Followers and Following -->
</div>
/* CSS */
.stats-container {
display: flex; justify-content: center; gap: 25px;
padding: 15px 0; border-top: 1px solid #eee;
border-bottom: 1px solid #eee; margin: 20px 0;
}
.stat-item { text-align: center; }
.stat-number { font-size: 1.3em; font-weight: bold; color: #667eea; }
.stat-label { font-size: 0.8em; color: #888; }

8 Add Skills CLASS

Add skill badges:

<!-- HTML - after stats -->
<div class="skills-container">
<span class="skill-tag">HTML</span>
<span class="skill-tag">CSS</span>
<span class="skill-tag">JavaScript</span>
</div>
/* CSS */
.skills-container { display: flex; flex-wrap: wrap; justify-content: center; gap: 8px; margin-bottom: 20px; }
.skill-tag {
background: linear-gradient(135deg, #667eea, #764ba2);
color: white; padding: 6px 14px; border-radius: 15px; font-size: 0.85em;
}

9 Add Social Links CLASS (Multiple Classes!)

Add social links with multiple classes:

<!-- HTML - after skills -->
<div class="social-container">
<a href="#" class="social-link github">💻</a>
<a href="#" class="social-link linkedin">💼</a>
<a href="#" class="social-link twitter">🐦</a>
<a href="#" class="social-link email">📧</a>
</div>
/* CSS */
.social-container { display: flex; justify-content: center; gap: 12px; }
.social-link {
width: 40px; height: 40px; border-radius: 50%;
display: flex; align-items: center; justify-content: center;
text-decoration: none; font-size: 1.1em;
}
.github { background: #333; }
.linkedin { background: #0077b5; }
.twitter { background: #1da1f2; }
.email { background: #e94560; }
